## How to run
Open `index.html` in a browser, or serve with any static server.

## Replace placeholder images
1) Put real images into `/assets/` (e.g. `/assets/kitchen-01.jpg`)
2) For gallery tiles on `projects.html`, add a `data-img="./assets/yourfile.jpg"` attribute to the `.tile` element.
3) For hero slides and service cards, replace the placeholder `.ph` / `.card-media` background with a real image:
   - Add `style="background-image:url('./assets/hero-01.jpg'); background-size:cover; background-position:center"` on the placeholder div.

## Minimum photo plan
- Home hero: 4 images (kitchen, ceiling, closet, tv panel/living)
- Each key section: 4 images as described in services page.
